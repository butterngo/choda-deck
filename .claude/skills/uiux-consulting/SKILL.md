# UI/UX Consulting

Design and review UI with a minimal, clean approach. Tailwind-based custom design system.

## Design philosophy

**Less is more.** Every element must earn its place.

## Core principles

| Principle | Rule |
|---|---|
| Clarity | User knows what to do within 3 seconds |
| Consistency | Same action = same visual pattern |
| Hierarchy | Most important action visually dominant |
| Feedback | Every action gets visible response |
| Progressive disclosure | Essentials first, details on demand |

## Anti-patterns

- Feature-stuffed screens (> 3 primary actions → split)
- Decorative elements with no function
- Inconsistent button styles
- No loading/error states
- Color-only feedback
