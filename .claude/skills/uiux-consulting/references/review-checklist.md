# UI Review Checklist

- [ ] Visual hierarchy: is the primary action obvious?
- [ ] Spacing: consistent rhythm, no cramped areas?
- [ ] Empty states: what does user see with zero data?
- [ ] Loading states: skeleton or spinner for async content?
- [ ] Error states: clear message + recovery action?
- [ ] Mobile: responsive or does it break?
- [ ] Accessibility: contrast ratio, keyboard nav, focus indicators?
